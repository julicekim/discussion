#### 🛠️ 보텍스(Vortex) API 실행 스크립트 (`commands/run_vortex_api.sh`)
- **기능**: `PYTHONPATH` 자동 설정, 모델 파일 존재 여부 실시간 체크, `uvicorn` 서버 기동을 한 번에 해결합니다!!
- **사용법**: 터미널에서 아래 명령어를 입력하기만 하면 됩니다.
  ```bash
  # 프로젝트 루트(anvil/)에서 실행!!
  ./commands/run_vortex_api.sh
  ```
