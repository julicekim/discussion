### 2. Vesper 트레이딩 엔진 이해도 (Core Strategy)
- **전략 명칭**: **Vesper Breakout** (장 초반 변동성 돌파 전략)
- **핵심 로직**:
    - **Entry**: 09:30~10:00 저항선 돌파, VWAP 상단 유지, 거래량 1.2배 폭발, 스코어링 모델(3.0 이상).
    - **Risk**: 변동성 3% 초과 시 차단, 슬리피지 0.05% 적용.
    - **Exit**: 1차 익절(50%), 본절 손절가(Break-Even), 장 마감 전 타임 컷.
- **기술적 진화**: 무거운 XGBoost Java 라이브러리를 제거하고, 덱스(Dex)가 구축한 **FastAPI(Vortex 1.0)**와 REST 통신(HTTP/1.1)을 통해 지능형 필터링을 수행합니다.

- **안정성 강화**: 티커 간 1.0초 대기 시간 적용 및 네트워크 오류/레이트 리밋 발생 시 자동 재시도 로직을 보강했습니다.
