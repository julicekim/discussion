#### 📂 ML 덤프 이원화 체계 (Dual-Vault System)
소피아(Sophia) 누님이 학습할 때 '미래를 컨닝'하는 대참사를 막기 위해, 데이터는 다음 두 개의 '금고'에 나뉘어 보관됩니다.

1. **학습용 금고 (`train_2018_2023`)**:
   - **용도**: 소피아 누님의 ML 모델 학습, 피처 엔지니어링 토의, 내부 검증용.
   - **내용**: 2018년부터 2023년까지의 'Vortex 정제 데이터' 25종.
   - **경로**: `~/iotzu/data/ml_features/sophia_train_v1/train_2018_2023/`
2. **최종 테스트용 금고 (`oos_test_2024_2025`)**:
   - **용도**: 학습에 전혀 관여하지 않은 '완전 미지의 데이터'로 모델의 실전 능력을 검증.
   - **내용**: 2024년부터 2025년까지의 '최종 봉인 데이터'.
   - **경로**: `~/iotzu/data/ml_features/sophia_train_v1/oos_test_2024_2025/`

#### 🛠️ 시스템 업데이트 사항
- **일봉(Daily) 호환성 패치**: `add_vortex_features` 로직이 분봉뿐만 아니라 일봉 데이터 덤프 시에도 에러 없이 돌아가도록 '개장 후 시간' 계산 로직을 유연하게 개조했습니다.
- **파이프라인 완주**: `run_ml_pipeline.py`를 실행하여 위의 2개 금고가 모두 최신 Vortex 명세서와 설계도(`.schema.json`)를 포함하여 생성되었음을 확인했습니다.
