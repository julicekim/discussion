#### 📊 훈련 요약 (Training Summary)
- **데이터셋**: 2018~2023년 25개 전 종목 데이터 병합 (**총 13,132,846행**)
- **모델 알고리즘**: XGBoost Classifier (Vesper Java 연동용 JSON 포맷)
- **소피아의 보수적 파라미터 적용**:
    - `max_depth=3`: 과적합(소설 쓰기) 방지
    - `scale_pos_weight=2.0`: 위험(1) 라벨 가중치 강화 (보수적 거래 차단 목적)

#### 📉 내부 평가 지표 (In-Sample Metrics)
훈련 데이터 내에서의 평가 결과는 다음과 같습니다:
- **정밀도 (Precision)**: 0.93 (라벨 1 기준)
- **재현율 (Recall)**: 1.00 (위험 상황을 놓치지 않으려는 소피아의 의지!!)
- **정확도 (Accuracy)**: 93%

#### 🧪 검증 결과 (Inference Test)
`tests/test_vortex_inference.py`를 통해 저장된 모델(`vortex_model_v1.json`)이 정상적으로 로드되고 추론을 수행하는지 확인했습니다.
- **결과**: 가상 데이터에 대해 0(안전) 또는 1(위험)을 100% 정상 예측 완료!!
